>## 정규표현식
>```
>문자열에서 패턴 검색
>```
>>리터럴(정적)
>>```angular2html
>>/패턴[ /플래그 ]
>>```
>>객체(동적)
>>```
>>new RegExp('패턴'[, '플래그']);
>
> 검색(반환값)
>>패턴 or null
>>```
>>패턴_변수.exec(문자_변수)
>>```
>>true or false ★★
>>```
>>패턴_변수.text(문자_변수)
>>```
>>패턴
>>```
>>문자_변수.match(패턴_변수)
>>```

+ 패턴
  >위치<br>
  >>``^``: 시작　　　``$``: 끝
  >
  >특수문자<br>
  >>``-`` : 범위<br>
  >>``|`` : OR  　　　 　　　　　``[]`` : OR(범위O)<br>
  >>``\d``, ``[0-9]`` :  숫자 (0~9)<br>
  >>``\w`` : 영어(a~Z) & 숫자(0~9)<br>
  >>``.`` : 임의의 문자 1개<br>
  >
  >반복
  >>``+`` : 패턴 1회 이상<br>
  >>``*`` : 패턴 0회 이상<br>
  >>``?`` : 패턴 0회 또는 1회 이상<br>
  >>``{n}`` : 패턴 n회<br>
  >>``{n,m}`` : 패턴 최소 n회, 최대 m회
  
+ 플래그
  + ``g`` : 모든 패턴 검색
  + ``i`` : 대·소문자 구분X 패턴 검색
  + ``m`` : 여러줄 문자열에서 패턴 검색
